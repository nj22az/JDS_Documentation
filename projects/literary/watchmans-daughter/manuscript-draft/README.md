# Standalone Draft

This directory is reserved for new or revised *Watchman's Daughter* prose.

Do not edit `source-canon/` or `manuscript-seed/`. When drafting begins, copy one
seed unit here, give it its intended standalone chapter number, and revise the
copy. This keeps every departure from the anthology visible and reversible.
