into this project because web use does not automatically establish print rights.

Su's established appearance must remain consistent with the existing character
reference: young Chinese-English woman, calm and watchful, with a restrained,
unreadable expression. New illustrations should extend that design rather than
recast her face.

Before print production, create a separate register containing creator, original
source, licence, credit line, resolution and permission status for every image.
