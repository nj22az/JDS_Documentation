# Figures — drop real images here

The book uses numbered figure placeholders. To replace a placeholder with a real
photo or illustration, drop a file named **`fig-0N.jpg`** (or `.png`) into this folder,
then rebuild the English edition. The build swaps the placeholder for the image
automatically; no manuscript edit needed.

| Figure | Filename to supply | Caption / brief | Used in |
|--------|--------------------|-----------------|---------|
| 1 | `fig-01.jpg` | Before & after — the same garage, cluttered (car outside) vs ordered (car in, pegboard wall) | Introduction |
| 2 | `fig-02.jpg` | The 10-step programme as one roadmap, grouped under the five 5S phases | The Programme |
| 3 | `fig-03.jpg` | The five-zone map — garage from above, five zones + car clearances | Ch 6 |
| 4 | `fig-04.jpg` | A shadow board — pegboard with tool outlines, one outline empty | Ch 8 |

- Numbering is fixed: Figure N ⇄ `fig-0N`. Add new figures by extending the list and
  using `[[FIGURE:N|caption]]` in the manuscript.
- Recommended: ~300 DPI for print; B&W or greyscale keeps the paperback in the cheap
  print tier. Landscape ~1800 px wide works well.
