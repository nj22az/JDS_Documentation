## Regulatory Basis

This program is prepared in accordance with **AFS 2017:3** — Use and Inspection of Pressurised Equipment (Arbetsmiljoverket), consolidated with amendments AFS 2019:1, AFS 2020:10, and AFS 2022:2.

---

## Program Contents

This binder contains the following documents, in order. Work through them from 01 to 07 when setting up a new program. Use documents 05 and 07 on a recurring basis.

| Doc | Title | Purpose | When to Use |
|:---:|-------|---------|-------------|
| **01** | Coordination Responsibility | Appoint the responsible person | Once at setup |
| **02** | Equipment Inventory | List and classify all vessels | Once, update as needed |
| **03** | Risk Assessment | Assess hazards per vessel | Once, review annually |
| **04** | Supervision Program | Define checks, frequencies, roles | Once, review annually |
| **05** | Supervision Round Record | Record each supervision round | Monthly / quarterly |
| **06** | Lifetime Journal | Track wall thickness, degradation | Per inspection cycle |
| **07** | Annual Review | Evaluate program effectiveness | Annually |

---

## How to Use This Program

1. **Print** all seven documents
2. **Fill in** documents 01-04 during initial site setup (one-time)
3. **Print multiple copies** of document 05 — you need one per round (12-17 per year)
4. **Fill in** document 05 each time you perform a supervision round
5. **Fill in** document 06 after each wall thickness measurement or inspection
6. **Fill in** document 07 once per year to review the program
7. **File** all completed documents in this binder, in chronological order
